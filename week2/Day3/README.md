# 🌗 Week 2 – Day 3: Theme Toggle (Dark/Light Mode)

## 📌  Overview

Built a Dark/Light Mode Toggle using JavaScript DOM manipulation. The theme changes dynamically and user preference is stored in localStorage, so it remains after page reload.

## 📚 Learned
DOM Manipulation
Event Handling with addEventListener
Class toggling using classList.toggle()
Using localStorage for state persistence
Dynamic UI updates based on user interaction
