# 🧠 Week 2 – Day 5: Quiz App (MCQs with Score Tracking)

## 📌 Overview
Built a Quiz Application that presents multiple-choice questions one at a time, tracks the user’s score, and displays the final result at the end. The app dynamically renders questions and options using JavaScript, captures user selections, validates answers, and updates the UI accordingly. It also includes functionality to restart the quiz and review progress through the session.

## 📚 Learned

Dynamic rendering of questions and options using DOM manipulation

Managing application state using variables (score, current question index, user answers)

Event handling for user interactions (button clicks)

Implementing score tracking logic based on user responses

Storing user-selected answers in an array

Using ES6+ features like arrow functions, template literals, and array methods (forEach)

Structuring data using arrays of objects for questions

Updating UI dynamically on each interaction

Resetting application state to restart the quiz

Understanding event-driven programming flow in JavaScript
