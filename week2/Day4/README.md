# 📝 Week 2 – Day 4: To-Do App with Local Storage

## 📌 Overview
Extended the To-Do App by adding persistent storage using localStorage. Tasks are saved and retrieved on page load, with features like due dates, task status (pending/completed), and delete functionality.

## 📚 Learned
LocalStorage for data persistence

JSON.stringify() and JSON.parse()

Storing and retrieving data from browser storage

Event handling for adding, deleting, and updating tasks

Dynamic UI updates based on stored data

Managing pending and completed tasks separately
