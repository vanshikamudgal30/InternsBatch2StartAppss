# 🧠 Week 3 – Day 1: React Basics (Counter App)

## 📌 Project Overview

This project is a simple Counter App built using React + Vite. It demonstrates basic React concepts like components, state management using useState, JSX, and event handling. The app allows users to increment and decrement a counter using buttons, with the UI updating dynamically on each click.

## 🚀 Tech Stack
React

Vite

JavaScript (ES6+)

CSS

## ⚙️ Installation & Setup

1. Create Vite App

npm create vite@latest

Select:

Framework: React
Variant: JavaScript

2. Install Dependencies

cd project-name

npm install

3. Run the App

npm run dev
