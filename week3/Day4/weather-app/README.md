# 🌤️ Week 3 – Day 4: useEffect Hook (Weather App)

## 📌 Project Overview

This project is a Weather App built using React that demonstrates the use of the useEffect hook for fetching data from an external API (OpenWeather API). The app fetches real-time weather information based on user input (city name) and displays it on the screen. It also handles loading states and errors, helping in understanding side effects, API calls, and conditional rendering in React.

## 🛠️ Tech Stack

React

Vite

JavaScript (ES6+)

CSS

OpenWeather API
