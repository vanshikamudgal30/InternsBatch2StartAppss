# 📐 CSS Flexbox & Grid Layout Project

## 🌐 Day 1 – Flexbox Basics & Layout Structure  
Developed responsive layouts using CSS Flexbox:

- Created a simple two-column layout using Flexbox  
- Worked with flex container and flex items  
- Practiced flex-direction for layout flow control  
- Used alignment properties like justify-content, align-items, and align-self  

### 📘 Learned:
- Flex container vs flex items  
- Flexbox layout model and behavior  
- Content alignment and spacing techniques  
- Responsive structure using Flexbox

Built a responsive navigation bar that adjusts based on screen size:

- Created horizontal navigation using Flexbox  
- Made navbar responsive for mobile and desktop views  
- Applied spacing and alignment for clean UI  
- Added hover effects for better user experience  

### 📘 Learned:
- Responsive design principles  
- Flexbox in navigation systems

## 🧮 Day 1 – CSS Grid Basics  
Created structured layouts using CSS Grid:

- Built a simple 3x3 grid layout  
- Used grid-template-columns and grid-template-rows  
- Controlled spacing using grid-gap  
- Positioned grid items using grid-column and grid-row  

### 📘 Learned:
- Grid container vs grid items  
- Two-dimensional layout system (Grid)  
- Precise layout positioning  
- Difference between Flexbox and Grid
